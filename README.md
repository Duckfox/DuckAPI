# DuckAPI
### 基于Bukkit的插件，作为我插件的前置插件使用
Gradle
```groovy
repositories {
    maven {
        name = "jitpack"
        url = "https://www.jitpack.io"
    }
}
dependencies {
    implementation 'com.github.Duckfox:DuckAPI:+'
}
```
Maven
```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://www.jitpack.io</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>com.github.Duckfox</groupId>
            <artifactId>DuckAPI</artifactId>
            <version>+</version>
        </dependency>
    </dependencies>
```
